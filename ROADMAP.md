# Роадмап

## ✅ В работе / готово частично

### Статистика в SQLite (`--stats`) — стартовала
`lib/stats.py` собирает в `data/aparser_stats.db`:
- **доменные зоны** из файлов результатов (URL по строке → суффикс через `tldextract`);
- **операторы запросов** из query-файлов (`site:.com`, `instreamset:(url):.org`, `inurl:…`).
Инкрементально: каждый результат разбирается один раз (по сигнатуре mtime:size).

Дальше по этому пункту:
- [x] снимки метрик заданий из карточек (status/done/total/failed%/speed/results) в ту же БД — time-series (`task_snapshots`), пишутся в проходе монитора для прогрессирующих заданий;
- [x] ретенция и оптимизация: удаление данных старше `stats_retention_days` (не чаще раза в 6 ч), WAL, индексы;
- [ ] интерфейс к статистике (вкладка в exe / веб-дашборд) — пока без интерфейса, только сбор;
- [x] топ-сводки в Telegram (`--top`): топ зон/операторов за `top_period_days` дней, `top_limit` позиций.

### Слежение за остатком и ETA — готово
Остаток = total − done по карточкам; скорость и ETA считаются по снимкам `task_snapshots`
(Δdone/Δвремя за окно `eta_window_min`, единицы-независимо). Уведомление «⏳ почти готово»
при `almost_done_pct`; остаток+ETA добавляются в сводку/heartbeat.
